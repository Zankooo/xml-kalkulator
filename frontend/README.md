## Nastavitev projekta
Naložiš node_modules in vse potrebne dependencies:
```sh
npm install
```

### Compile kode in predogled v browserju

```sh
npm run dev
```
Klikneš na localhost link, ki se prikaže v konzoli

### Compile and Minify for Production

```sh
npm run build
```
